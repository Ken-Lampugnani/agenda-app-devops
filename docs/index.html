Agenda App

DevOps-Projekt – Phase "Build"

Projektbeschreibung

Die Agenda App ist eine einfache Webanwendung zur Verwaltung von Traktanden. Benutzer können Einträge hinzufügen und wieder entfernen. Das Projekt dient zur praktischen Umsetzung der DevOps-Phase "Build".

Funktionen

  • Hinzufügen neuer Traktanden
  • Entfernen bestehender Einträge
  • Dynamische Aktualisierung der Liste
  • Einfache und benutzerfreundliche Oberfläche

Technologien

  • SvelteKit – Frontend-Framework
  • JavaScript (JSDoc)
  • npm als Paketmanager
  • Git & GitHub für Versionskontrolle
  • GitHub Pages für Deployment

DevOps Build-Phase

Das Projekt wurde lokal entwickelt, versioniert und anschliessend über GitHub Pages veröffentlicht. Dadurch wurde ein einfacher und stabiler Deployment-Prozess umgesetzt.
