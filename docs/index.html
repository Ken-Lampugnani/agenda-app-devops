Agenda App

Kleine Traktandenliste zum Hinzufügen und Entfernen von Einträgen.

Projektziel

Eine einfache Agenda-/Traktandenliste, um grundlegende SvelteKit-Konzepte und den DevOps-Build-Prozess zu üben.

Features

  • Traktandum hinzufügen
  • Traktandum entfernen
  • Einfache, klare UI

Tech Stack

  • SvelteKit
  • JavaScript (JSDoc)
  • npm
